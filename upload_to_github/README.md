# Redick2019.github.io — DFIR & Life Labs Portfolio

Personal portfolio website for a Digital Forensics & Incident Response graduate.

Live: https://Redick2019.github.io

## Stack
- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS v3
- Framer Motion (animations)
- GitHub Actions (auto-deploy on push)

## Local Development
```bash
npm install
npm run dev
```

## Deploy
Push to main branch — GitHub Actions builds and deploys automatically.
